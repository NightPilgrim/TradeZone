# TradeZone — Telegram Web App

Минималистичное приложение для учёта сделок в Telegram.

## Структура

- `index.html` — разметка, депозит, список сделок, график, кнопки
- `style.css` — стили в духе Telegram, адаптив, сворачивание, анимации
- `app.js` — логика: WebApp API, localStorage, баланс, Chart.js, CRUD, экспорт

## Запуск локально

Раздайте папку по HTTP (Telegram Web App открывается по HTTPS или `http://localhost` в dev).

Пример через Python:
```bash
cd trading-journal
python -m http.server 8080
```
Откройте в браузере: `http://localhost:8080`

## Подключение к Telegram

1. Создайте бота через [@BotFather](https://t.me/BotFather), команда `/newbot`.
2. В настройках бота: **Menu Button** или **Web App** — укажите URL вашего приложения (например `https://your-domain.com/trading-journal/` или ngrok для локальной разработки).
3. Открывайте бота в Telegram — откроется Web App.

## Функции

- **Депозит** — ввод и сохранение начального депозита (localStorage).
- **Сделки** — дата/время, актив (BTC/USD, ETH/USD, EUR/USD, Gold или свой), результат (±), размер позиции (опционально).
- **Сворачивание** — каждая сделка сворачивается/разворачивается по клику на заголовок (▶/▼).
- **Баланс** — депозит + сумма результатов, зелёный/красный цвет.
- **График** — линейный график баланса (Chart.js), точки по хронологии сделок.
- **Фильтр** — по активу (включая свои варианты из сделок).
- **Очистить историю** — удаление всех сделок с подтверждением, депозит не трогается.
- **Экспорт в JSON** — выгрузка депозита и сделок.
- **Поделиться результатом** — итоги в чат/буфер.
- **Тема** — подхватывается из Telegram (светлая/тёмная).
